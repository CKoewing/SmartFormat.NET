# Contributing

## Issues

[Open an issue](https://github.com/scottrippey/SmartFormat.NET/issues?state=open) for anything you would like to see in SmartFormat.NET, but please check other issues first.

## Pull Requests

Pull requests are welcome.

Please submit to the `master` branch. Don't forget to add unit tests since we're not accepting untested changes.
